About

Hi! I’m Gaurav Bhalla (as I assumed you’d guessed by now), and this is my personal website. It’s a collection of my projects, and a struggling blog (ThinKode) that I’ve wrote one post for. This website will be neglected and updates will be sporadic at best, so when they do come out, be sure to visit them. I’m probably going to continue with some JavaScript projects before I figure out how to display my projects in other languages like Python. If you want to test any of my projects on your own computer, just download them using the download icon next to the project description. If you know how to run a localhost (which you need to run them), you should be fine.

If you don’t, however, you can always just use the online version. Any project ideas I have will hopefully be blogged about, and I’m working on uploading my code with explanation to my blog.

I hope you can find something interesting here, and if you have any suggestions, post them on Reddit, here. I will take them into account, and any inappropriate suggestions will be removed, with repeat offenders banned.

My favourite youtube channel is sentdex, he's an absolute legend and where I learnt to code first in python. I hope I can also interest some people that visit my website to pick up coding, whether for personal interest or for money.
