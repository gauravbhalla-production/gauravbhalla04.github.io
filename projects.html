Projects

This is a page for all my projects, it includes a link to the projects online, a download link, and a brief documentation about the project. I hope you find this interesting, and are able to get something useful from this page.

As of 26/3/2020, all my projects on this website are made in JavaScript using the p5js library, this is because these are the most easily shared projects due to their compatibility with websites. I'm working on a solution to display my other projects in python - the other main language that I work with.

In addition to the four original projects being displayed here, I'm working on writing up the documentation for each and uploading it, I will be uploading them all together, not one by one. I aim to get these proposals fullfilled by the end of 2020, and once my plans have been completed, I'm sure I will have more ideas to display on this page as well as more improvements that can be made to the original projects.

To run a project on your own computer, you can use the previously mentioned download links and run a localhost server on your computer to view them, or feel free to use an online IDE like the p5js editor. If you happen to stumble across a service that offers support for embedded python projects, please let me know via one of the social media platforms listed in the navbar.
